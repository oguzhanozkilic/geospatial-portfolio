\## Projects



\### Delaware Investment Potential Score Analysis

Census tract-level investment scoring for Delaware using official US federal data.



\*\*Pipeline:\*\*

\- Data: US Census TIGER/Line 2023 + ACS 5-Year 2022

\- Database: Supabase (PostgreSQL + PostGIS)

\- Scoring: HUD AMI-based income scoring + vacancy rate + median home value

\- Output: Choropleth map (QGIS) + GeoJSON



\*\*Tools:\*\* Python · GeoPandas · SQLAlchemy · PostGIS · QGIS



\*\*Data Sources:\*\* US Census Bureau · HUD AMI Standards
